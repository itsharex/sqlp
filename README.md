<h1>sqlp</h1>
> A tool for quickly viewing CSV

### 截图

![image](/demo/screen1.png)

### 安装依赖

```sh
pnpm i
```

### 运行

```sh
pnpm tauri:dev
```

### 打包

```sh
pnpm tauri:build
```

### 图标生成
* 修改public文件夹下的app-icon.png图标为实际项目图标，格式为1024x1024px的png，然后执行下面的命令即可一键生成所有平台的icon并放在src-tauri/icons文件夹中
pnpm icon
